Echo Button Hybrid API Test

🔄 Hybrid API Approach

This test verifies that the EchoButton component uses a hybrid API approach:
  • HTML Attributes: Use kebab-case (`icon-position`, `icon-size`, `icon-variant`)
  • JavaScript Properties: Use camelCase (`iconPosition`, `iconSize`, `iconVariant`)
  • Main Property: Simplified to just `icon` (no kebab-case needed)

HTML Usage (Attributes)

<echo-button icon="plus" icon-position="left">Add</echo-button> <echo-button icon="settings" icon-size="large">Settings</echo-button> <echo-button icon="star" icon-variant="filled">Star</echo-button>

JavaScript Usage (Properties)

button.icon = 'plus'; button.iconPosition = 'left'; button.iconSize = 'large'; button.iconVariant = 'filled';
