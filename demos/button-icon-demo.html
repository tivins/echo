Echo Button with Icons Demo

Basic Icon Buttons

Basic buttons with icons positioned on the left (default) or right.
Add Item Delete Edit Save
<echo-button icon-name="plus" context="primary">Add Item</echo-button> <echo-button icon-name="trash" context="danger">Delete</echo-button> <echo-button icon-name="edit" context="secondary">Edit</echo-button> <echo-button icon-name="save" context="success">Save</echo-button>

Icon Position

Icons can be positioned on the left (default) or right side of the button text.
Back Next Download Upload
<echo-button icon-name="arrow-left" icon-position="left" context="primary">Back</echo-button> <echo-button icon-name="arrow-right" icon-position="right" context="primary">Next</echo-button> <echo-button icon-name="download" icon-position="left" context="success">Download</echo-button> <echo-button icon-name="upload" icon-position="right" context="info">Upload</echo-button>

Different Sizes

Icons automatically scale with button size, or you can override the icon size.
Extra Small Small Medium Large
<echo-button icon-name="star" size="xs" context="warning">Extra Small</echo-button> <echo-button icon-name="star" size="small" context="warning">Small</echo-button> <echo-button icon-name="star" size="medium" context="warning">Medium</echo-button> <echo-button icon-name="star" size="large" context="warning">Large</echo-button>

Different Variants

Icons inherit the button's context color and work with all button variants.
Default Outline Ghost Soft Link
<echo-button icon-name="heart" variant="default" context="danger">Default</echo-button> <echo-button icon-name="heart" variant="outline" context="danger">Outline</echo-button> <echo-button icon-name="heart" variant="ghost" context="danger">Ghost</echo-button> <echo-button icon-name="heart" variant="soft" context="danger">Soft</echo-button> <echo-button icon-name="heart" variant="link" context="danger">Link</echo-button>

Custom Icon Properties

You can override icon size and variant independently from the button.
Small Icon Medium Icon Large Icon Filled Icon
<echo-button icon-name="settings" icon-size="small" context="primary">Small Icon</echo-button> <echo-button icon-name="settings" icon-size="medium" context="primary">Medium Icon</echo-button> <echo-button icon-name="settings" icon-size="large" context="primary">Large Icon</echo-button> <echo-button icon-name="settings" icon-variant="filled" context="primary">Filled Icon</echo-button>

Disabled State

Icons respect the button's disabled state.
Enabled Disabled Enabled Disabled
<echo-button icon-name="check" context="success">Enabled</echo-button> <echo-button icon-name="check" context="success" disabled>Disabled</echo-button> <echo-button icon-name="x" context="danger">Enabled</echo-button> <echo-button icon-name="x" context="danger" disabled>Disabled</echo-button>

Icon Only Buttons

Buttons can contain only icons without text.
<echo-button icon-name="plus" context="primary"></echo-button> <echo-button icon-name="edit" context="secondary"></echo-button> <echo-button icon-name="trash" context="danger"></echo-button> <echo-button icon-name="heart" context="danger"></echo-button> <echo-button icon-name="star" context="warning"></echo-button> <echo-button icon-name="settings" context="info"></echo-button>
